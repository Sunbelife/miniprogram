##已实现接口说明文档

### 登录部分

- 一览：
  - /verify_user

- verify_user - 验证用户登录：

```
api: /verify_user

请求方式：get
请求参数：
{
  user_name:"admin",
  pass_word:"admin"
}
返回：
{
  "code": 200,  // 成功 200，失败 250
  "msg": "验证成功",
  "data": "null"
}
```
