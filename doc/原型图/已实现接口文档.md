## 微信小程序

### 用户登录

- 一览：
  - /verify_user
  - /get_union_id
- verify_user - 验证用户：
  - 记录到服务器 time 和 session_key
  - open_id, union_id, time, session_key

```
api: /verify_user

请求方式：get
请求参数：
{
	code:"xxx", // 临时登录凭证code
}
返回：
{
	"code": 0,  /* -1	系统繁忙，此时请开发者稍候再试
					  200	请求成功
					  40029	code 无效
					  45011	频率限制，每个用户每分钟100次 */
	"msg": "获取成功",
	"data":
	{
		"open_id":"xxx"
	}
}
```

- get_union_id - 得到 union_id

留着 union_id 就完成了

```
api: /get_union_id

请求方式：get
请求参数：
{
	open_id:"xxx",
	iv:"xxx",
	encrypted_data :"xxx" // 加密串
}
返回：
{
	"code": 200,  /* 成功 200
					 41001: encodingAesKey 非法
					 41003: aes 解密失败
					 41004: 解密后得到的buffer非法
					 41005: base64加密失败
					 41016: base64解密失败 */
	"msg": "获取成功",
	"data": 
	{
		null
	}
}
```
