## 微信小程序

### 用户登录

- 一览：
  - /verify_user
  - /get_union_id
- verify_user - 验证用户：
  - 记录到服务器 time 和 session_key
  - open_id, union_id, time, session_key

```
api: /verify_user

请求方式：get
请求参数：
{
	code:"xxx", // 临时登录凭证code
}
返回：
{
	"code": 0,  /* -1	系统繁忙，此时请开发者稍候再试
					  200	请求成功
					  40029	code 无效
					  45011	频率限制，每个用户每分钟100次 */
	"msg": "获取成功",
	"data":
	{
		"open_id":"xxx"
	}
}
```
